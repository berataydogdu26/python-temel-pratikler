# 🐍 Python Temel Algoritma ve Otomasyon Pratikleri

Bu depo, Python öğrenim sürecimde geliştirdiğim 3 farklı komut satırı uygulamasını ve pratik kod örneklerini içermektedir.

## 📦 Proje İçerikleri

### 1. Öğrenci Kayıt & Sorgulama Otomasyonu (`01_ogrenci_otomasyonu.py`)
- **Teknolojiler:** Nested Dictionaries, `while` döngüsü, `if-elif-else`
- **İşlev:** Öğrenci numarasına göre verileri ezmeden kayıt ekler ve sorgular.

### 2. Vize/Final Not Hesaplayıcı (`02_not_hesaplama.py`)
- **Teknolojiler:** Aritmetik Operatörler, Koşullu İfadeler
- **İşlev:** Vize (%60) ve Final (%40) ağırlıklarını hesaplayarak geçme/kalma durumunu belirler.

### 3. Kullanıcı Kayıt ve Giriş Paneli (`03_giris_paneli.py`)
- **Teknolojiler:** Dictionaries (Sözlükler), Kimlik Doğrulama Mantığı
- **İşlev:** Kullanıcı adı ve şifre verilerini eşleştirerek güvenli giriş simülasyonu sağlar.
